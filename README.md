4-DOF Robotic Arm Simulation

A Python-based desktop application that simulates a 4 Degree of Freedom robotic arm using Forward Kinematics.

This project was developed as the Computer Programming II (CP-II) semester project for the Department of Mechanical, Mechatronics & Manufacturing Engineering.

Features
2D Robotic Arm Simulation
Interactive GUI
Forward Kinematics Visualization
Real-time Joint Angle Adjustment
User-friendly Control Panel
Educational Tool for Robotics Learning
Technologies Used
Python
Tkinter
NumPy
Matplotlib
